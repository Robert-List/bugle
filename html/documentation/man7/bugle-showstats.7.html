NAME

showstats - render statistics over the display  

SYNOPSIS

filterset showstats
{
    show "statistic1"
    show "statistic2"
    graph "statistic3"
}
 

DESCRIPTION

The showstats filter-set displays statistics gathered by other filter-sets over the rendered display. These filter-sets must be listed in the chain ahead of showstats. Statistics can be displayed either as text or as graphs.

The statistics to display are selected by the show and graph pseudo-variables in the filters file. These reference statistics that are defined in $HOME/.bugle/statistics. Most users should just copy this file from the example in the BuGLe documentation; it lists most of the statistics that one might wish to use.  

OPTIONS

time
The time interval between updating the statistics display.
key_accumulate
By default, statistics are averaged over the time interval specified above, making them approximately instantaneous rates. Pressing this key will cause the display to instead show the average from the time the key was pressed until the current moment.
key_unaccumulate
This key restores the default behaviour of showing instantaneous rate.
 

FILES

$HOME/.bugle/statistics
Formulae and metadata for statistics
 

ENVIRONMENT

BUGLE_STATISTICS
If set, defines the location of the statistics file instead of $HOME/.bugle/statistics.
