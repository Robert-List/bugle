NAME

frontbuffer - force rendering to the front buffer  

SYNOPSIS

filterset frontbuffer
 

DESCRIPTION

The frontbuffer filter-set overrides the draw buffer, setting it to GL_FRONT. This is most useful when used within gldb(1), in order to see how a scene is built up batch by batch.  

BUGS

As with any filter-set that modifies OpenGL state, it will break complex rendering techniques like render-to-vertex-buffer.

Rendering to off-screen buffers (such as pbuffers) and multiple render targets are not tested and may fail.
